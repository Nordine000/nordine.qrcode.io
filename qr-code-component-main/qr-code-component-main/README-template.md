## Mon processus

### Construit avec

- **HTML5 sémantique**
- **CSS personnalisé**
- **Flexbox** pour le centrage
- **Google Fonts** (Outfit, Manrope, Montserrat, Playwrite NO)
- **Responsive Design** (viewport et unités relatives)

### Ce que j’ai appris

- Comment utiliser **Google Fonts** via `<link>` et `font-family`.  
- Structurer une carte avec **Flexbox** pour centrer verticalement et horizontalement.  
- Utiliser des **variables HSL** pour gérer les couleurs de manière cohérente.  
- Ajouter un favicon et gérer la responsivité avec la balise `<meta name="viewport">`.

Exemple de code dont je suis fier :

```html
<main>
  <section>
    <img src="images/image-qr-code.png" alt="QR code">

  </section>
</main>
